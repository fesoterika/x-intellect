# X-Intellect — восстановленный сайт проекта

Новый сайт проекта **X-Intellect** (ранее — «Сфера Разума», основан в 2012 г. А. Г. Глазом):
Laravel-приложение с админкой, публичной частью на архивных мотивах, аудиоплеером,
глоссарием с тултипами и SEO-автоматизацией. Цель проекта — сохранить материалы
архива сайта (статьи, вики, аудиозаписи) в удобном для чтения виде.

## Стек

- **Laravel 13** (PHP 8.3), Blade — серверный рендер, SEO из коробки
- **SQLite** (`database/database.sqlite`) — вся база материалов хранится прямо в репозитории
- **Alpine.js** — аудиоплеер, тултипы глоссария; **Trix** — редактор админки; **Vite + Tailwind** — сборка фронтенда
- **Database-очередь + cron** (`queue:work --stop-when-empty`) — фоновые задачи без воркер-процесса

## Что реализовано

| Раздел | Реализация |
|---|---|
| Модель данных | `sections`, `pages` (seo JSON, `source_type` — эпохи контента, ревизии), `media`, `glossary_terms`, `redirects`, `menu_items` |
| Admin CMS | `/admin` (auth + роли admin/editor, rate-limit логина, noindex), CRUD всего, Trix с кнопкой `[[audio:ID]]` |
| Публичная часть | космическая тема, бейджи эпох, хлебные крошки, вики-сайдбар, таймлайн истории, дисклеймер в футере |
| Аудиоплеер | Alpine-компонент: плейлист, скорость, перемотка ±15с, позиция в localStorage; short-код `[[audio:ID]]` |
| SEO | авто-slug (транслитерация), meta из seo-полей, JSON-LD (Article/Person/FAQPage/BreadcrumbList), `sitemap:generate`, middleware редиректов (301 архивные / 302 `/go/*`) |
| robots/llms | статичные `public/robots.txt`, `public/llms.txt` |
| Глоссарий | автопростановка терминов в тексте статей (тултипы), страница `/glossary` |
| Импорт архива | локальный офлайн-слепок сайта разобран и залит: основной сайт, вики, термины глоссария, аудио, перелинковка внутренних ссылок |

## Локальная разработка

```bash
composer install
cp .env.example .env && php artisan key:generate
php artisan migrate                                 # база уже содержит материалы (database.sqlite в репозитории)
php artisan storage:link
npm install && npm run dev                          # или npm run build
php artisan serve
```

Пароль администратора задаётся через `.env` (`SEED_ADMIN_EMAIL`/`SEED_ADMIN_PASSWORD`)
и меняется после первого входа на `/profile`.

Тесты: `php artisan test` (59 тестов: публичные страницы, редиректы, роли, SEO, ревизии).

## Аудиозаписи

Аудиофайлы архива (сеансы, приветствия — суммарно ~1,9 ГБ) не хранятся в этом
репозитории — отдельные записи слишком велики для git. Полный комплект аудио
выложен в облаке:

**https://cloud.mail.ru/public/sUfY/jpDWsSH5N**

Для локальной работы с плеером файлы нужно скачать и разложить по путям,
указанным в таблице `media` (`storage/app/public/media/audio/archive/`).

## Импорт архива

Команды разбора локального офлайн-слепка сайта (`app/Console/Commands/`):

- `import:offline-explorer {archive}` — основной сайт → страницы разделов (черновики)
- `import:offline-wiki {wiki-dir}` — вики → страницы «Вики» + термины в «Глоссарий»
- `import:offline-audio {wiki-dir}` — привязка аудио к сеансам и «Приветствию»
- `remap:archive-links` — перелинковка внутренних ссылок на актуальные адреса нового сайта

Все команды идемпотентны и поддерживают `--dry` (просмотр без записи) и,
где применимо, `--refresh` (перечистка тела уже импортированных страниц).
Импортированные страницы создаются черновиками — публикация вручную после вычитки.

## Безопасность

- `.env` не в git (секреты — `APP_KEY`, пароли, токены)
- Rate-limit на `/login`, публичная регистрация отключена
- `composer audit` / `npm audit` перед релизом
