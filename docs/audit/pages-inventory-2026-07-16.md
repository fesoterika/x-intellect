## Причина недостачи

Импортёр `import:offline-wiki`, `import:offline-audio` и аудит `audit:archive` перечисляют файлы
только в корне `wiki/` (`File::glob($base."/index.php@title=*")`). Offline Explorer разложил часть
страниц по переполненным подпапкам `%&Ovr0`…`%&Ovr119` — эти страницы были невидимы и для импорта,
и для аудита (отсюда ложное «172/172»). Пример: `%&Ovr3/index.php@title=…Сеанс_с_силами_20070730b`.
Основной сайт: `ImportOfflineExplorer` смотрит только глубину 1 (`glob("*/default.htm")`).
